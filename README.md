# Movie Poster App

## Features
- Swipeable gallery of movie posters.
- Detailed view for each movie poster, including:
  - **Title**
  - **Rating**
  - **Author**
  - **Description**
- Option to add selected posters to a watchlist with real-time updates.
- Custom app icon and splash screen with the title.
